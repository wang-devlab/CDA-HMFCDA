# circRNA-GAT Feature Enhancement Project

This project enhances circRNA sequence features using Graph Attention Networks (GAT) for improved disease association prediction.

## Project Structure
project/
├── data/ # Input data files
├── models/ # Neural network models
├── utils/ # Utility functions
├── outputs/ # Output files and results
├── main.py # Main execution script
├── requirements.txt # Python dependencies
└── README.md # Project documentation

## Installation
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Place your data files in the `data/` directory

## Usage
Run the main script:

```bash
python main.py
